# Desafio Classificador de Nível de Herói - DIO

Projeto desenvolvido como desafio da DIO.

##  Tecnologias utilizadas
- JavaScript
- Node.js

## Objetivo
Classificar o nível de um herói com base na quantidade de XP.

##  Níveis
- Ferro
- Bronze
- Prata
- Ouro
- Platina
- Ascendente
- Imortal
- Radiante

## Exemplo de saída

O Herói de nome Jungkook está no nível de Platina
